# Docs References

## Goal
沉淀最近做微信小程序 AI 接入过程中积累的可复用知识，放在 `docs/references/` 下作为"开发经验参考"，供未来其它项目查阅。

## Scope
- 通用知识，**不涉及**本项目业务逻辑
- 中文写作，面向中文微信小程序开发者
- 每篇文档独立成章，可单独迁移

## Deliverables
1. `docs/references/README.md` — 索引页
2. `docs/references/wechat-miniprogram-ai-integration.md` — wx.cloud.extend.AI 完整指南
3. `docs/references/wechat-miniprogram-streaming-output.md` — 流式输出 4 种方案
4. `docs/references/wechat-miniprogram-backend-options.md` — 后端架构选型
5. `docs/references/wechat-miniprogram-js-runtime.md` — JS 运行时限制

## Non-Goals
- 不是入门教程，假定读者知道小程序基本概念
- 不覆盖 WXML/WXSS 等 UI 相关知识
- 不讨论本项目 ring-miniApp 的实现细节
